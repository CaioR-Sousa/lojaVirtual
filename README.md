# 🛒 Loja Virtual

Uma aplicação web de uma **Loja Virtual** desenvolvida para praticar conceitos de desenvolvimento Front-End. O projeto simula uma experiência de compra online, permitindo visualizar produtos, adicioná-los ao carrinho e acompanhar o valor total da compra.

## 🚀 Funcionalidades

* 🛍️ Catálogo de produtos
* 🔎 Exibição de informações dos produtos
* ➕ Adicionar produtos ao carrinho
* ➖ Remover produtos do carrinho
* 💰 Cálculo automático do valor total
* 📱 Layout responsivo para diferentes dispositivos
* 🎨 Interface moderna e intuitiva

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
